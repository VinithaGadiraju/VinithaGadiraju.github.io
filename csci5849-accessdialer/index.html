Help

This application includes a dialer, a contact list, an add contact form, and a help tab. The website includes the following accessibility features:

  • Tabbing through all buttons and forms
  • The application has keyboard shortcuts to navigate between tabs and for the buttons on the dialer. Depending on your system, the keyboard shortcut method will be different.

    If you are using a Mac on chrome, firefox, or safari, hold down the control key and then the option key before pressing the keyboard shortcut.

    If you are using Windows or Linux with a chrome browser, hold down the alt key before pressing the keyboard shortcut.

    If you are using Windows or Linux with firefox, hold down the alt key and then the shift key before pressing the keyboard shortcut.

    If you are using Windows on Safari, hold down the alt key before pressing the keyboard shortcut.

    The following keyboard shortcuts are to navigate between tabs: press (d) for Dialer, (l) for Contact List, (a) for Add Contact, and (h) for Help.

  • In the dialer, each number on the keyboard corresponds to the button on the interface. For example, pressing 1 on the keyboard will add 1 to the phone number in the input form. For the pound sign, press (p) and for the star press (s). The numbers included on the dialer are 0 through 9. To clear the form, press (c). To dial, press (e).
  • In the add contact page, press (n) to add contact and (f) to clear the form. Again, use the keyboard shortcut method outlined above depending on your browser and operating system.
  • The contact page is formatted as an accessible table with the columns name, phone number, email address, and contact image. Each contact image includes a short caption and a longer description.
  • The application has ARIA landmark roles and regions.
